# Styles Directory

Global styles and design system for the AI learning platform.

## Design System Structure

```
/styles
├── globals.css         # Global styles and CSS reset
├── variables.css       # CSS custom properties for theming
├── components/         # Component-specific styles
├── layouts/           # Layout-specific styles
└── utilities/         # Utility classes
```

## Design Principles
- **Educational Theme**: Clean, focused design that doesn't distract from learning
- **High Contrast**: Excellent readability for long-form content
- **Accessibility**: Color contrast ratios, focus indicators, reduced motion support
- **Mobile-First**: Responsive design with touch-friendly interactions
- **Dark/Light Mode**: Theme switching for different learning environments

## Technology Options
- **TailwindCSS**: Utility-first framework for rapid development
- **CSS Modules**: Component-scoped styles
- **Styled Components**: CSS-in-JS for dynamic theming
- **Sass/SCSS**: Enhanced CSS with variables and mixins

The styling approach will be determined based on the framework selection.
